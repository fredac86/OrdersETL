#Customer Order ETL

ETL em Pyspark que faz a extração de dados de CVS's contido no Azure Blob Storage, efetua algumas transformações de interesse do negócio e faz ocarregamento da tabela final no Azure SQL server para consulta.

[EN] ETL in Pyspark that transmits data from CVS's contained in Azure Blob Storage, generating some transformations of interest to the business and loading the final table on the Azure SQL server for query.
